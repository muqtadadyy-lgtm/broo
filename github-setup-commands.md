# GitHub Setup Commands for Main Project

After creating your new GitHub repository, run these commands:

1. Remove the incorrect remote:
```bash
git remote remove origin
```

2. Add your new repository remote (replace YOUR_USERNAME and REPO_NAME):
```bash
git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git
```

3. Push to GitHub:
```bash
git push -u origin master
```

# git remote add origin https://github.com/muqtadadyy-lgtm/brooo.git
